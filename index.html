How to use this catalog

Technology Platforms

Core facilities and infrastructure available across RARE.Qc member institutions. Browse by domain, type, or institution.

What is a Core Facility?

A shared research facility offering specialized equipment and expertise to multiple research groups, typically with fee-for-service or tiered pricing.

What is a Data Resource?

A dataset, database, or biobank accessible to researchers—including patient registries, genomic databases, or clinical data repositories.

What is Expertise?

Consultation services from specialized researchers or staff—no physical equipment. Ideal for methodological guidance or bioinformatics support.

Updating this catalog

Upload an updated Excel file to Claude and ask to regenerate the data. Then replace this HTML file on GitHub. No code changes needed.
